# Proof Gold Website

## Project Overview
Static marketing website for the Proof Gold trading EA, hosted on GitHub Pages.

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Hosting:** GitHub Pages (static)
- **No build tools** - all files served directly

## Project Structure
- `index.html` - Main landing/marketing page
- `account.html` - User account management
- `affiliates.html` - Affiliate program info
- `brokers.html` - Supported brokers list
- `setup.html` - EA installation guide
- `troubleshooting.html` - Common issues and fixes
- `vps.html` - VPS setup guide
- `chat-widget.html` - Support chat integration
- `risk.html`, `terms.html`, `privacy.html`, `cancellation.html` - Legal pages
- `sitemap.xml` - SEO sitemap
- `robots.txt` - Crawler directives
- `logo.svg` - Brand logo

## Development
- No build step required - edit HTML files directly
- Preview locally by opening HTML files in a browser
- Deploy by pushing to main branch (GitHub Pages auto-deploys)

## Agent Activation
Specialized agents are available in `~/.claude/agents/`:
- **Frontend Developer** - For website UI and content work
- **Full Stack Developer** - For cross-repo changes
- Activate with: "activate Frontend Developer mode"

## Guidelines
- Keep pages lightweight and fast-loading
- Ensure mobile responsiveness on all pages
- Update `sitemap.xml` when adding new pages
- Maintain consistent branding and styling
- Test across browsers before pushing
